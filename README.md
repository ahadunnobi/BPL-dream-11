**BPL Dream 11** is a high-performance fantasy cricket management application built with the latest web technologies. Experience the thrill of scouting, selecting, and managing your own Bangladesh Premier League (BPL) squad with a sleek, dark-themed interface and real-time state management.

## ✨ Core Features

- **🏆 Dynamic Player Scouting**: Browse an extensive list of BPL stars with detailed information including role, country, and market value.
- **💰 Credit Management System**: Claim free credits via the interactive banner to fund your team-building journey.
- **🛡️ Squad Selection**: Select and manage players for your dream team. The system tracks your squad size and remaining budget in real-time.
- **🌗 Premium UI/UX**: Built with a "Dark Mode first" philosophy using Tailwind CSS 4 and DaisyUI 5, featuring glassmorphism and smooth micro-animations.
- **🔔 Real-time Notifications**: Integrated with `react-toastify` for instant feedback on team selections and credit additions.
- **📱 Fully Responsive**: Optimized for all devices, from desktop monitors to mobile screens.

## 💻 Local Installation

Follow these steps to get the project running on your local machine:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/ahadunnobi/BPL-dream-11.git
   ```

2. **Navigate to the Project Directory**
   ```bash
   cd BPL-dream-11
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Launch Development Server**
   ```bash
   npm run dev
   ```

5. **Open in Browser**
   The application will be running at `http://localhost:5173`.
